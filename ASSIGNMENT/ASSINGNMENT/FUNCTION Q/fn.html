What are functions?

Functions in JavaScript are reusable blocks of code that can be defined and invoked to perform a specific task.
They are a fundamental concept in programming and play a crucial role in structuring and organizing code. In JavaScript
functions are reusable blocks of code that can be defined with the
function keyword. They encapsulate a set of instructions and can take parameters,
perform a task, and return a value. Functions facilitate code organization, reusability, and modularity in JavaScript programs.

How are functions implemented in JS?

Function Declaration:

function greet(name)
console.log('Hello');

Parameters:

Functions can take parameters, which act as variables within the function. Parameters are listed inside the parentheses.

function greet(name) { console.log('Hello, ' + name + '!'); }

Return Statement:

Functions can return a value using the return statement. The function stops execution when a return statement is encountered.

function add(a, b) { return a + b; } var result = add(3, 5); // result will be 8

Function Expressions:

Functions can also be assigned to variables, creating function expressions.

var multiply = function(a, b) { return a * b; };

Anonymous Functions:

Functions can be created without a name, known as anonymous functions. Often used in function expressions or as arguments to other functions.

var add = function(x, y) { return x + y; };

Arrow Function:

Introduced in ECMAScript 2015 (ES6), arrow functions provide a concise syntax for writing functions.

var square = (num) => num * num
