What are loops?

for loop:

The for loop is used when you know in advance how many times you want to iterate.
It consists of three parts: initialization, condition, and iteration statement.

for (let i = 0; i < 5; i++) { console.log(i); }

while loop:

The while loop repeats a block of code as long as a specified condition is true.

let i = 0; while (i < 5) { console.log(i); i++; }

do-while loop:

Similar to the while loop, the do-while loop executes a block of code at least once,
and then it repeats as long as a specified condition is true.

let i = 0; do { console.log(i); i++; } while (i < 5);

for-each loop:

In JavaScript, the forEach loop is a method for iterating over elements in an array.
It provides a more concise and readable way to iterate through each element compared to traditional for loops.
The forEach method is available on arrays and takes a callback function as its argument.
This callback function is executed once for each element in the array.

const array = [1, 2, 3, 4, 5]; array.forEach(function(element) { // code to be executed for each element console.log(element); });

What is an arrow function?

Arrow Function:

Arrow functions in JavaScript provide a concise syntax for writing anonymous functions,
with implicit return for single expressions, and lexically bind the this keyword.

array.forEach((element, index, array) => { console.log(`Element at index ${index}: ${element}`); });

What is map function in js?

Map Function:

In JavaScript, the map function is an array method that creates a new array by applying a provided function to
each element of the original array. It doesn't mutate the original array. The result is an array with the same length, where each element
corresponds to the result of the applied function on the original element.
